# pubsub-transport-tcp

Internal implementation detail of the [`pubsub`](https://crates.io/crates/pubsub) crate.

This crate provides the TCP transport for the pubsub message broker. It is not intended for direct use. Depend on [`pubsub`](https://crates.io/crates/pubsub) instead, with the `server` feature enabled if you need to embed a broker.
