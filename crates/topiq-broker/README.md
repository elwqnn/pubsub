# topiq-broker

Internal implementation detail of the [`topiq`](https://crates.io/crates/topiq) crate.

This crate provides the broker engine for the topiq message broker. It is not intended for direct use. Depend on [`topiq`](https://crates.io/crates/topiq) instead, with the `server` feature enabled if you need to embed a broker.
